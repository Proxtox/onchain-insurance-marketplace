# On-Chain Insurance Marketplace

Decentralized insurance marketplace with parametric products, peer-to-pool, and automated claims.

## Features
- Parametric insurance products
- Peer-to-pool insurance
- Automated claims processing
- Risk pricing engine
- Multi-protocol coverage

## Architecture
- Insurance marketplace contracts
- Parametric trigger modules
- Claims and payout engine
- Risk and pricing layer
